# Python Learning Repository

This repository contains Python programs and exercises created while learning core Python concepts. It includes basic scripts, practice problems, and small projects to build a strong foundation in Python programming.

## Topics Covered

* Variables and Data Types
* Operators and Expressions
* Conditional Statements
* Loops
* Functions
* File Handling
* Lists and Dictionaries
* Arrays and NumPy Basics

## Folder Structure

* `01_module.py` → Basic module usage
* `02_practice_set1.py` → Practice problems
* `03_variables.py` → Variables and data types
* `08_loops.py` → Loop concepts
* `10_FileHandling.py` → File operations
* and more...

## How to Run

1. Clone the repository:

   ```bash
   git clone <your-repo-url>
   ```
2. Navigate to the folder:

   ```bash
   cd PYTHON
   ```
3. Run any file:

   ```bash
   python filename.py
   ```

## Requirements

* Python 3.x
* Some programs may require external libraries (e.g., `pyttsx3`)

Install dependencies:

```bash
pip install pyttsx3
```

## Purpose

The goal of this repository is to track progress, practice coding, and improve problem-solving skills in Python.
